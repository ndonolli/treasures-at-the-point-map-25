# Treasures at the Point 2026 Map

Static map for browsing Treasures at the Point 2026 listings in Algiers Point.  I usually update the local data every year based off the flyer that gets circulated.  This is hooked up to a netlify deploy pipeline that watches the main branch and deploys.  It's just a static site after all.

The site is hosted at [https://treasures-at-the-point-map.netlify.app/](https://treasures-at-the-point-map.netlify.app/)
